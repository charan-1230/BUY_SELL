# Buy-Sell @ IIITH

##  Tech Stack
- *Frontend:* React.js (Tailwind CSS for styling)
- *Backend:* Express.js
- *Database:* MongoDB
- *Server Runtime:* Node.js
- *Authentication:* JWT
- *Hashing:* bcrypt.js for password hashing

---

## Installation Guide
###  *Download the Zip*

   Download the Zip file and Extract it.
   (you may clone the repo instead)


### *Front-End Setup*

sh 

    cd frontend

    npm i

    cd ..

    cd backend

    npm i

    cd ..

    cd frontend 

    npm run dev

open new terminal

sh

    cd backend 

    npm run dev
